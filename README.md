# MealMate 🍽️ – Recipe Finder Web App

## 📌 Description

MealMate is a web application that allows users to search and explore recipes using a public API. It provides a clean and user-friendly interface to discover meals, view detailed recipes, and organize results efficiently.

## 🔌 API Used

TheMealDB API  
https://www.themealdb.com/api.php

## ✨ Features

- 🔍 Search recipes by name  
- 🥗 Filter recipes by category (e.g., Vegetarian, Dessert, Seafood)  
- 📊 Sort recipes alphabetically  
- ❤️ Add recipes to favorites (stored in localStorage)  
- 🌙 Dark/Light mode toggle  
- 📱 Fully responsive design  

## 🛠️ Tech Stack

- HTML  
- CSS (or Tailwind/Bootstrap)  
- JavaScript (ES6)  
- Fetch API  

## 🚀 Setup Instructions

1. Clone the repository  
2. Open index.html in your browser  
3. Start exploring recipes!
